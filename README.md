# Sector ETF Holdings Aggregator

This repository contains a simple Python script that downloads Excel
holdings files for U.S. sector ETFs and prints comma-separated lists of
stock tickers held by each fund.

## Usage

1. Create a virtual environment and install dependencies:

   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```

2. Add ETF symbols and their corresponding daily holdings URLs to the
   `ETFS` dictionary in `aggregator.py`.

3. Run the script:

   ```bash
   python aggregator.py
   ```

   Example output:

   ```text
   Fetching holdings for XLV...
   XLV: JOHNSON & JOHNSON,PFIZER,MERCK & CO INC,...
   ```

## Extending

- Additional ETFs can be added by extending the `ETFS` mapping.
- The script currently looks for a column whose name includes
  "ticker"; modify `tickers_from_df` if your files are structured
differently.

## Simple Dark-theme Website

A lightweight static site lives in the `site/` directory. It loads
`holdings.json` (generated by running the script with `--json`) and
renders each ETF's tickers on a dark background. To update the data,
run the script with `--json` and commit the resulting JSON file.

Vercel can serve the `site` directory directly; a minimal `vercel.json`
can be added if you want to control the output folder:

```json
{
  "public": "site"
}
```

## Notes

The ETFs used by the US sectors are typically provided by State Street
Global Advisors (SSGA) and the URLs change daily; be sure you are using
the correct link for each fund.
